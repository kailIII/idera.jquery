idera.jquery Un plugin de jQuery para visualizar los datos de IDERA.

Audiencia

Esta página está diseñada para gente familiarizada con el lenguaje de programación Javascript y con la librería jQuery.

Requerimientos

  • idera.jquery depende de la librería jQuery.
  • La versión mínima de jQuery que necesita es 1.7.2 pero se recomienda la última versión.
